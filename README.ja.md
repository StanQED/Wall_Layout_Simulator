# Wall Layout Simulator

[English](README.md) | 日本語

Wall Layout Simulator は、部屋の壁面レイアウト、壁面アート、家具、ドア、窓枠の配置をブラウザ上で設計・確認するためのスタンドアロン Web アプリケーションです。

単一の `index.html` として実装されており、ビルドやサーバーを用意しなくても、ブラウザで直接実行できます。

## 主な機能

- 3D 部屋レイアウトのプレビュー
- 前面・背面・左面・右面の 2D 壁面ビュー
- 上面ビュー、3D ビュー、カメラリセット
- 部屋サイズの変更
- 壁色・床色の変更
- 壁面アートの追加
  - 額装
  - ポスター
  - 布・タペストリー
  - 鏡
- 用紙・グッズサイズのプリセット
  - A 判、B 判
  - 色紙
  - ロングタペストリー
  - 抱き枕カバー
  - 正方形
  - カスタムサイズ
- 家具の追加
  - ソファ
  - テーブル
  - 棚
  - 収納
  - ベッド
- ドア・窓枠オブジェクトの追加
- オブジェクトの選択、複製、削除、ロック、非表示、再表示
- 壁面オブジェクトの整列・均等配置
- 壁面アートへのローカル画像の貼り付け
- URL からの画像読み込み
- JSON によるシーンの保存・読み込み
- PNG 画像としての書き出し
- 日本語・英語 UI の切り替え
- ガイドツアー
- デモシーン
  - リビング例
  - ギャラリー例

## 基本的な作業手順

1. 部屋の幅、奥行き、高さを設定します。
2. 配置対象の壁を選択します。
3. 壁面アート、家具、ドア、窓枠を追加します。
4. オブジェクトを移動、整列、複製、削除します。
5. 必要に応じて PNG または JSON として書き出します。
6. 保存した JSON を読み込むことで、後から編集を再開できます。

## キーボードショートカット

| キー | 操作 |
|---|---|
| `G` | 選択オブジェクトを移動 |
| `R` | 選択オブジェクトを回転 |
| `S` | 範囲選択 |
| `V` | 選択解除 |
| `Delete` / `Backspace` | 選択オブジェクトを削除 |
| `Shift + D` | 選択オブジェクトを複製 |
| `Ctrl + Z` | 元に戻す |
| `Ctrl + Y` | やり直し |
| `1` | 前面壁の 2D ビュー |
| `7` | 上面ビュー |
| `5` | 投影方式の切り替え |
| `M` | 左右パネルの表示・非表示 |
| `?` | ガイドツアーを開始 |
| `F` | ズームとパンをリセット |
| `Esc` | 現在の操作をキャンセル |

## JSON データ

シーンは JSON ファイルとして書き出し、再度読み込むことができます。

JSON には主に次の情報が含まれます。

- 部屋のサイズと色
- オブジェクト一覧
- オブジェクトの位置とサイズ
- 選択中の壁
- カメラ状態
- レイヤーと表示設定

## 画像読み込みに関する注意

ローカル画像はファイル選択から壁面アートに貼り付けられます。

外部 URL の画像は、ブラウザの CORS 制約の影響を受ける場合があります。URL 画像を読み込んだ後に PNG 書き出しが失敗する場合は、ローカル画像を使用してください。
